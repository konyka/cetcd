cetcd — implementation notes

Pure-C reimplementation of etcd. Living progress log. Last updated: Phase 1 complete.

Status

Phase 1 (IO + protobuf) complete. 16/16 tests passing with ASan+UBSan+LeakSan. Vendored libuv, libco (cloudwu/coroutine), protobuf-c runtime. libcetcd_io (stubs for loop/TCP/timer/coroutine API) and libcetcd_proto (KV/RPC/Auth message structs + wire codec) implemented. Ready for Phase 2 (Raft core).
