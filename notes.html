cetcd — implementation notes

Pure-C reimplementation of etcd. Living progress log. Last updated: Phase 4 complete — MVCC + Lease, 23/23 tests.

Status

Phase 4 (MVCC + Lease) complete. 23/23 tests passing. Treap ordered map with range queries. MVCC store with revision-based indexing, historical reads, watch (exact key + prefix), and lease manager with TTL min-heap. Next: Phase 5 — HTTP/2 + gRPC + TLS.
